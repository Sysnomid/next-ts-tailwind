# NextJS-TS-Tailwind Boilerplate

NextJS Boilerplate with Typescript and Tailwind

**To use this Boilerplate**:
    <br />
    ``` npx nextjs-ts-tailwind```
    <br />
    ``` cd name_of_folder && npm install ```
    Then edit ``` pages/index.tsx ```
